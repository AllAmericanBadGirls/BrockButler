How can I plan my day?

To add a task
Click on the menu button and select Add Task. In the Add Task window, select the course you wish to add the task to from the drop down menu.

To modify/remove a task
Expand the course that contains the task you wish to remove. Then click the white i to modify, or the red x to delete the task. To mark a task as completed, click the checkbox next to the task.

To set task priority
You can set a task's priority during creation, or you can change its priority by modifying the task.
